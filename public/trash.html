How to create a feed?

All you need to do is to select blocks that you want to see in your feed. We will use this pattern to fetch data from this site and put it in your feed.

1. Select a basic News item block

This block is required to set content borders for each news item. You should select only a single news article when specifying a news item. FetchRSS will search for similar blocks on this page and take them as separate news to include them into your feed. All news components (like title, content, etc.) will be picked from the inside of this block.

2. Select a Headline

After you've specified a news item, now it's time to pick its components.
Headline is a header of the news. This block is required in order to generate a valid RSS.
If you are familiar with RSS file structure you can consider it as <title>...</title> block

3. Select a Summary

Summary is a main content of the news. It's a news body.
If you are familiar with RSS file structure you can consider it as <description>...</description> block

You can continue to specify other news components. Like "author", "illustration" and so on. All other blocks are optional. And can be added to make your feed more detail.

If you don't want to specify some component or need to change previous selection just click on a relevant component button.

After you've finished specifying your feed content click "Generate RSS" button and your feed is ready.
